Image processing code using NumPy and Pandas

The code works for binary images.

Finds the skew in the light portion of the image (rectangular objects) and finds the correction angle.

The correction angle is the angle by whic the light rectangular object needs to rotated to align it in a right position.

To run the code:

- you nust have NumPy and Pandas installed in your system.
- use the command: "python image.py"
